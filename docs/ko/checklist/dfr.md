# 4.1.7. 데이터 흐름 제한 점검 체크리스트

데이터 흐름 제한 점검 체크리스트는 로봇 시스템 내·외부 간 데이터 흐름을 통제하기 위한 것으로, 네트워크 분리, 필터링, 최소 데이터 전달 원칙 등을 포함합니다. 불필요하거나 위험한 정보의 외부 유출을 방지하는 것을 목표로 합니다.

| 번호 | 보안 수준 | 체크리스트 항목 | 내용 |
| :--- | :--- | :--- | :--- |
| DFR-01 | L1 | 네트워크 기능<br>지원 | 구역과 통신경로의 분할을 위해 운영자가 네트워크 설정을 직접 관리·설정할 수 있는 기능을 제공해야 한다. |
| DFR-02 | L2 | 프로토콜 필터링 | 승인된 프로토콜만 허용하고 승인되지 않은 프로토콜의 사용을 제한하는 기능을 제공해야 한다. |
