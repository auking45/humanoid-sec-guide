# 4.1.8. Event Response Checklist

The Event Response Checklist is for identifying and responding to security breaches or abnormal behavior. It includes log management, alarms, diagnostics, and incident response procedures, aiming to secure records necessary for detecting and forensically investigating security incidents.

| ID | Security Level | Checklist Item | Description |
| :--- | :--- | :--- | :--- |
| ER-01 | L1 | Audit log accessibility | The capability for authorized personnel and/or tools to access audit logs in a read-only manner shall be provided. |
| | L3 | Programmatic access to<br>audit logs | Programmatic access to audit logs using an API shall be provided. |
| ER-02 | L2 | Continuous monitoring | The capability for continuous monitoring to detect and report security violations in a timely manner shall be provided. |
| ER-03 | L2 | Security event notification | The capability to send notifications to administrators through various channels such as email, SMS, mobile app push notifications, voice alerts, or on-screen displays when critical security events occur shall be provided. |
