# 4.1.7. Data Flow Restriction Checklist

The Data Flow Restriction Checklist is for controlling data flow between the inside and outside of the robot system. It includes network segmentation, filtering, and the principle of least data transfer, aiming to prevent the external leakage of unnecessary or dangerous information.

| ID | Security Level | Checklist Item | Description |
| :--- | :--- | :--- | :--- |
| DFR-01 | L1 | Network function support | The capability for operators to directly manage and configure network settings for the segmentation of zones and communication paths shall be provided. |
| DFR-02 | L2 | Protocol filtering | The capability to allow only authorized protocols and restrict the use of unauthorized protocols shall be provided. |
